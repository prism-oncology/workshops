# Spatial omics practical workshop

> [!NOTE]
> Workshop given at the *Mathematical Foundations for Spatial Omics in oncology summer school* (20-24 July, 2026).

This workshops introduces several spatial omics tools:
1. [Sopa](https://github.com/prism-oncology/sopa) for data preprocessing.
2. [scConcept](https://github.com/theislab/scConcept) for cell-type annotation.
3. [Novae](https://github.com/prism-oncology/novae) for spatial domains annotation.

## Requirements

Both tutorials are independant, and their dependencies are listed at the beginning of the notebook.

All dependencies can be installed on all OS, for any **Python version `>=3.11`**.

> [!TIP]
> We recommend having one environment per notebook.
